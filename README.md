# Основные квантовые алгоритмы
### Реализация квантовых алгоритмов, с использованием библиотек Cirq, QisKit. 

## Реализованы следующие алгоритмы:
- Алгоритм Дойча
- Алгоритм Дойча-Йожи
- Алгоритм Бернштейна-Вазирани и алгоритм Саймона
- Реализация проктокола BB84
- Состояние Белла и квантовая телепортация
- Алгоритм оценки фазы
- Квантовое преобразование Фурье (QFT)
- Алгоритм Китаева
- Наивный алгоритм 
- Алгоритм Шора
- Алгоритм поиска порядка
- Алгоритм Гровера
